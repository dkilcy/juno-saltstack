
### Network

##### Switch setup

1. Connect the USB console cable
2. Start minicom: `minicom -D /dev/ttyUSB0 -b 38400`
3. Configure switch
```
enable

```

Other commands:
```
show vlan brief
show vlan id 1

show interface status 
show interface vlan 1             

show system-info
show ip http secure-server
```
